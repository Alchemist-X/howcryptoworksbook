## 一、数字所有权的基本问题

2021 年 3 月，Beeple 的 "Everydays" 在佳士得拍出 **$7,000 万**。买家 Metakovan 买到的不是图片的独占访问权——任何人都可以右键保存同样的像素。他买到的是一个**密码学验证的"原件"证明**。

这引出了 NFT 要解决的根本问题：**在完美复制零成本的数字世界中，如何创造"所有权"？**

### NFT 的四层所有权结构

NFT 的核心创新是**将数字资产的所有权拆解为可独立编程的多个层次**：

```
┌─────────────────────────────────────────────┐
│  层 1：代币控制权                             │
│  区块链不可变地记录谁控制 NFT #1234           │
├─────────────────────────────────────────────┤
│  层 2：来源证明                              │
│  可追溯至创作者钱包，建立真实性                │
├─────────────────────────────────────────────┤
│  层 3：使用权                                │
│  独立的许可证（通常在链下）定义持有者能做什么   │
├─────────────────────────────────────────────┤
│  层 4：实用访问权                             │
│  智能合约根据代币持有状态授予权限（门控功能等）  │
└─────────────────────────────────────────────┘

每一层可独立编程，具备传统资产不具备的灵活性：
· 可随时间演变  · 可路由版税  · 可与其他数字资产交互  · 甚至可控制自己的钱包
```

### 唯一性的技术实现

同质化代币（ERC-20）如同相同面值的纸币，NFT 则如同编号的宝可梦卡片：

```
ERC-20：Alice 持有 100 USDC ← 每个 USDC 完全等价
ERC-721：Alice 持有 NFT #1234 ← 全球唯一（合约地址 + 编号的组合）
```

更新的创新：**NFT 可以拥有自己的钱包**。一个游戏角色 NFT 可以在你玩的过程中积累装备和货币。卖出这个 NFT 时，它拥有的一切自动随之转移。

### 版权困境：两条路线之争

NFT 证明的是代币持有权，但底层作品的使用权由**链下许可证和传统版权法**管辖——这在去中心化资产语境下产生了巨大的灰色地带。

这催生了 NFT 世界最根本的路线分歧：

| 路线 | 代表项目 | 理念 |
|------|---------|------|
| **封闭品牌** | Bored Ape Yacht Club | 授予持有者商业权利，但保留核心 IP 控制 |
| **CC0 公共领域** | Nouns DAO、CrypToadz | "无权利保留"——任何人可使用、混编、商业化 |

CC0 的核心论点：**品牌像协议一样运作——当它开放且无许可时，价值反而更大。**

## 二、超越静态图片：NFT 的进阶形态

### 存储方案的取舍

NFT 面临一个根本的技术两难：把内容存在链上（永久但昂贵），还是链下（便宜但可能失效）？

| 存储方案 | 持久性 | 成本 | 风险 |
|---------|-------|------|------|
| 中心化服务器 | 低 | 最低 | 服务器关闭 → NFT 指向死链接 |
| IPFS | 中 | 低 | 需持续"pin"才能保持可用 |
| Arweave | 高 | 中（一次性付费永久存储） | 依赖捐赠模型长期可行性 |
| 链上存储 | 最高 | 最高（单张图片可能数千美元 Gas） | 真正的抗审查和永久性 |

成熟的项目采用分层策略：IPFS/Arweave 哈希确保文件不可静默篡改 + 关键来源信息直接上链 + 多个 pin 服务商备份。

### 高级代币类型

**动态 NFT：** 响应外部数据而变化——球星卡自动更新赛季数据、数字艺术根据天气变色、游戏角色随经验值升级而改变外观。

**可组合 NFT：** 代币嵌套代币。一块虚拟世界的土地（NFT）包含一栋房子（NFT），里面有家具（更多 NFT）。出售土地时，一切随之转移。

**半同质化代币：** 模糊了同质与非同质的边界。活动门票开始时完全相同（可互换），使用后变为唯一的（记录座位、入场时间）。

**灵魂绑定代币（SBT）：** 故意设计为**不可转让**——用于身份、证书、成就或声誉。大学学位 NFT 不应该可以卖给别人。

### NFT 的主要类别

```
PFP 项目（头像类）
├── CryptoPunks、Bored Apes、Pudgy Penguins
├── 数字身份符号 + 社交信号
├── 类比：物理世界的劳力士 = 数字世界的 Bored Ape 头像
├── 网络效应：名人采用 → 主流认知 → 文化价值
└── 现实：Bored Ape 地板价较 ATH（>$400,000）跌幅超 90%

生成艺术
├── Art Blocks 等平台上的算法生成作品
├── Tyler Hobbs 的 Fidenzas、Snowfro 的 Chromie Squiggles
└── 价值来自艺术性、历史意义和算法创新

游戏 & 虚拟世界
├── Axie Infinity 的生物、The Sandbox 的地块
├── "Play-to-Earn" 叙事强大但多数项目经济模型不可持续
└── 难以在初始投机之后留住玩家

实用/访问权 NFT
├── 作为数字钥匙：社区、活动、软件、服务
└── 正被探索用于忠诚度计划和订阅模型

身份/证书 NFT
├── 可验证的文凭、认证、执照
└── 灵魂绑定代币确保不可转让
```

> 蓝筹 NFT（CryptoPunks、Bored Apes）在市场崩塌后仍保持文化地位和有意义的地板价，而长尾项目几乎全部归零——这与传统艺术和收藏品市场的规律完全一致。

### 供给机制

| 机制 | 原理 | 效果 |
|------|------|------|
| 固定供应 | 如 10,000 个 CryptoPunks，永不增发 | 绝对稀缺性 |
| 联合曲线（Bonding Curve） | 每次铸造价格递增（如从 0.1 ETH 起，每枚增加 0.01） | 奖励早期参与者 |
| 燃烧机制 | 永久销毁代币（如燃烧 3 个普通换 1 个稀有） | 通缩压力 |

## 三、技术基础

### ERC-721：NFT 的基础标准

ERC-721 定义了非同质化代币在智能合约层面的运作方式。核心函数出奇简洁：

```solidity
ownerOf(tokenId)           // "谁拥有 NFT #1234？"
transferFrom(from, to, id) // "将 NFT #1234 从 Alice 转给 Bob"
approve(to, tokenId)       // "Alice 授权 Bob 转移她的 NFT #1234"
setApprovalForAll(op, ok)  // "Alice 授权市场转移她的所有 NFT"
```

### ERC-1155：多代币标准

```
ERC-721：一个合约 = 一类 NFT
ERC-1155：一个合约 = 同质化 + 非同质化代币的混合管理
  · 批量操作：一笔交易转移数十个代币（大幅节省 Gas）
  · 游戏场景的理想选择：独特武器 + 可互换金币
```

### 安全风险与常见骗局

`setApprovalForAll` 等授权函数赋予了强大但危险的权限：

- **钓鱼攻击：** 伪装成官方网站或 Discord/X 消息，诱导用户连接钱包进行"免费铸造"或"空投"
- **钱包清空器：** 诱导用户签署看似合法的交易，实际授予攻击者清空所有资产的权限

> 核心自保原则：硬件钱包存储高价值资产，"一次性"钱包用于新项目铸造。

### 发行策略

| 策略 | 优势 | 劣势 |
|------|------|------|
| 公平发行（先到先得） | 对所有人公平 | 机器人主导 |
| 荷兰拍卖（高价起步递减） | 抗机器人操纵 | 定价不确定 |
| 白名单（Allowlist） | 奖励社区早期贡献者 | 可能被操纵 |
| 联合曲线 | 价格随供需调整 | 晚期参与者成本高 |

### Solana NFT 生态

Solana 发展出了独立于以太坊的 NFT 生态系统：

- **Metaplex 框架：** Solana NFT 的基础设施标准
- **压缩 NFT（Compressed NFTs）：** 利用状态压缩技术，数百万 NFT 铸造成本仅几分钱——解锁大规模空投、忠诚度计划和游戏资产等用例
- **市场格局：** Magic Eden（用户友好）和 Tensor（专业交易者工具）主导；不同于以太坊的 OpenSea 长期垄断
- **代表项目：** Solana Monkey Business、Mad Lads、Claynosaurz

## 四、NFT 在哪里交易：市场战争

### OpenSea vs. Blur：版税之战

NFT 市场的竞争围绕一个**技术缺陷**展开：**创作者版税从未内置于 ERC-721/1155 核心标准**。ERC-2981 允许合约"建议"版税金额，但支付完全是自愿的。

```
时间线：
2022.11  OpenSea 推出 Operator Filter，强制新项目收取版税
         但无法对旧项目强制执行
              ↓
2023 初  Blur 上线：版税可选（最低 0.5%）+ 专业工具 + BLUR 代币激励
              ↓
2023.02  Blur 交易量超过 OpenSea
              ↓
2023.08  OpenSea 被迫放弃版税强制政策
              ↓
2024-25  Blur 激励减退 + NFT 交易量全面崩塌
         → OpenSea 重新夺回以太坊 NFT 市场主导地位
```

**聚合器的角色：** Gem 和 Genie 等聚合协议跨市场比价，自动将用户路由到最低成本的平台——这无形中放大了 Blur 的费用优势。两者很快被收购（Gem → OpenSea，Genie → Uniswap），后来随着交易集中回 OpenSea 而逐渐失去相关性。

### NFT 定价的独特挑战

与同质化代币不同，NFT 的价格发现是**逐件进行**的：

| 概念 | 含义 | 注意事项 |
|------|------|---------|
| **地板价（Floor Price）** | 一个系列中最便宜的 NFT 价格 | 基线指标，但可能严重低估稀有个体价值 |
| **特征定价（Trait-based）** | 根据稀有属性（金色皮毛、激光眼等）单独定价 | 更精确，但实际采用率低 |
| **集合竞价（Collection Bid）** | "任何满足条件的 NFT 我都要" | 改善流动性，但将"独特"资产商品化 |

> NFT 的核心悖论：每一个设计决策都在**独特性**与**可替代性**之间摇摆。地板价交易将独特的艺术品当商品对待；特征竞价试图量化不可量化的审美；集合竞价为流动性牺牲了独特性。这个张力定义了 NFT 的一切。

## 章节要点速览

1. **NFT 不是卖图片——而是将数字所有权拆解为代币控制、来源证明、使用权和实用访问四个可编程层次**
2. **"右键保存"不影响价值：NFT 的价值来自可验证的稀缺性和来源证明，如同原作与复制品的区别**
3. **存储是生死问题：链上最持久但最贵，中心化服务器最便宜但可能让 NFT 变成"死链接"**
4. **动态 NFT、可组合 NFT 和灵魂绑定代币将 NFT 从静态收藏品扩展为活跃的数字基元**
5. **蓝筹项目保持价值，长尾归零——NFT 市场结构与传统艺术/收藏品市场高度一致**
6. **版税从未内置于标准中，导致市场竞争围绕"谁少收费"展开，最终损害了创作者收入**
7. **Solana 的压缩 NFT 技术将铸造成本降至几分钱，为大规模应用场景打开了大门**
