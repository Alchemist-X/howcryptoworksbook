## 一、DeFi 的核心理念

如果说 Bitcoin 的目标是创造不依赖任何权威的货币，DeFi 的野心更大：**如果我们能创造一个没有银行、经纪商和清算所的平行金融体系呢？**

DeFi 建立在无许可区块链上，任何人都可以使用、审计和在其上构建。与传统金融每一层都依赖中介不同，DeFi 将金融逻辑直接编码进智能合约。

### DeFi 的独特属性

| 属性 | 说明 |
|------|------|
| 全天候运行 | 没有收盘时间，没有假日 |
| 原子可组合性 | 借款→兑换→存入——可在单笔交易内完成，要么全部成功，要么全部回滚 |
| 透明可验证 | 每笔交易和协议规则都可公开审计 |
| 货币乐高 | 协议像积木一样拼接，创造传统金融不可能的创新 |

### 基本权衡

DeFi 不是免费午餐。用户面临：Gas 费、滑点、MEV 提取（第八章详述）、无常损失、授权风险（恶意代币可通过无限授权耗尽资金）、智能合约 Bug 和预言机故障。

> DeFi 独特地奖励那些深谙协议行为的技术专家——许多 MEV 机会直接源于协议机制。

## 二、去中心化交易所（DEX）架构

### Uniswap：AMM 革命

传统订单簿需要持续的挂单/撤单和毫秒级撮合——在 Ethereum 的低吞吐量和 12 秒出块时间下不可行。Uniswap 用**自动做市商（AMM）**取代了撮合引擎：用数学公式从池子余额直接报价。

#### 恒定乘积公式：直觉解释

```
池子包含两种代币 X 和 Y：
  x × y = k（常数）

想买 X？→ 向池子中加入 Y，取出 X
约束：乘积 k 不变

结果：
· 买得越多，每多买一个的价格越高（指数级递增）
· 池子永远不会被买空——数学上不允许
· 池子越深（流动性越大），每笔交易的价格影响越小
```

**直觉类比**：想象两桶弹珠（红色和蓝色），有一条魔法规则：红色数量 x 蓝色数量 = 固定值。想买红弹珠就得加蓝弹珠。桶越来越"吝啬"——第 1 个便宜，第 50 个极贵。桶越深（弹珠越多），每次取出的影响越小。

#### Uniswap 的演化

| 版本 | 核心创新 | 意义 |
|------|---------|------|
| **v1** | ETH 路由——所有代币对都通过 ETH | 概念验证 |
| **v2** | 任意 ERC-20 对 + TWAP 预言机 + 闪电兑换 | 通用化 |
| **v3** | 集中流动性——LP 选择特定价格区间 | 资本效率革命 |
| **v4** | 单合约架构 + Hooks（可编程 AMM 行为） | 动态费率、MEV 感知流等 |

**集中流动性**（v3）特别适合高度相关资产（稳定币对、stETH/ETH），LP 可以在已知公允价值附近集中资本，大幅降低滑点。LP 还可以通过只在当前价格上方或下方提供流动性来实现渐进买入或卖出（范围订单）。

### Curve Finance：稳定币交易的数学

Uniswap 的恒定乘积公式对稳定币交易效率低下——理论上 1:1 交换的资产，流动性却分散在极少发生的价格范围。

```
Curve 的 StableSwap 曲线：

在锚定附近（~1:1）→ 近似恒定和（极低滑点）
偏离锚定时       → 逐渐过渡到恒定乘积（防止池子崩溃）

放大因子 A 控制曲线的"平坦程度"：
· A 越高 → 在锚定附近滑点越低
· 费率仅 0.01-0.04%（vs Uniswap 的 0.3%）
```

**3pool**（USDC/USDT/DAI）成为稳定币基础设施的支柱。Meta-pool 让新稳定币直接与 3pool LP Token 配对，无需碎片化流动性即可接入三大稳定币。

**竞争格局转变**：Uniswap v3 的 0.01% 费率层 + 集中流动性逐渐侵蚀了 Curve 的市场份额。Uniswap 现在日均处理超 2.2 亿美元 USDC/USDT 兑换，Curve 全部稳定币池约 4400 万美元。

### Bonding Curve 发射台：Pump.fun

```
代币生命周期（Pump.fun 模式）：

创建代币（固定供应，如 10 亿枚）
    │
    ▼
Bonding Curve 阶段
  · ~8 亿枚放上曲线，用 SOL 买卖
  · 价格随购买量非线性上升
  · 创建者可在最低价格处优先购买
    │
    ▼
达到毕业门槛（目标 SOL 数量）
    │
    ▼
自动毕业
  · 系统在 AMM（Raydium 或 PumpSwap）创建流动性池
  · 绑定的 SOL + 代币注入池中
  · 通常燃烧 LP Token
    │
    ▼
在 AMM 上正常交易
```

**现实**：只有 ~1-2% 的代币能毕业。大多数在曲线上作为短命的社交实验死去。尽管如此，Pump.fun 成为 2024-2025 周期最具影响力的零售应用之一，将代币发行成本和摩擦降至几乎为零。

### 替代交易架构

| 架构 | 代表 | 核心机制 |
|------|------|---------|
| **意图系统** | CoW Swap, UniswapX | 用户签署意图，求解器竞争最优执行 |
| **RFQ 系统** | Hashflow, 0x RFQ | 专业做市商链下报价，链上结算，零滑点 |
| **去中心化永续** | GMX, dYdX, Hyperliquid | 链上杠杆交易——从合成定价到专用应用链 |

## 三、借贷协议

### Aave：自动化借贷基础设施

Aave 就像一个永不关门的自动化银行。核心概念：

```
存入 ETH → 获得 aToken（余额自动增长）→ 赚取利息
  │
  ▼
抵押 ETH → 借出 USDC（超额抵押）
  │
  │  健康因子（HF）= 抵押品价值 / 债务价值
  │  HF > 1 = 安全
  │  HF < 1 = 可被清算
  │
  ▼
利率根据池子利用率自动调整
  · 利用率高 → 利率上升（吸引存款、抑制借贷）
  · 利用率低 → 利率下降
```

**超额抵押**是无信任借贷的基石——协议不能起诉违约者或扣押工资，必须有足够资产可供清算。

#### 使用场景

| 场景 | 逻辑 |
|------|------|
| 不卖获流动性 | ETH 持有者借 USDC 用于开支——保留上行潜力、延迟资本利得税 |
| 杠杆循环 | 存 ETH → 借 USDC → 买更多 ETH → 重复（激进者 HF 低至 ~1.2） |
| 做空/对冲 | 借入预期下跌的资产并立即卖出——链上主经纪商功能 |
| 套利与套息 | 借入便宜的稳定币，在别处赚取更高收益 |

#### 版本演化

| 版本 | 核心改进 |
|------|---------|
| v1 | 池化借贷 + 闪电贷 |
| v2 | 债务代币化 + 信用委托 + 抵押品置换 |
| v3 | 隔离模式（安全上线长尾资产）+ 效率模式（相关资产更优利率）|
| v4（开发中）| 统一流动性层——中央流动性枢纽 + 资产专用辐射 |

### Euler 与 Morpho：隔离的无许可市场

与 Aave 的池化蓝筹设计不同，Euler 和 Morpho 推向了更模块化的方向：

**Euler v2**：Euler Vault Kit（EVK）——任何人可为任意 ERC-20 部署隔离借贷金库，自定义参数。金库之间的问题不会互相感染。

**Morpho Blue**：极简无信任借贷原语——一种贷款资产、一种抵押品、一个清算 LTV、一个预言机、一个利率模型。在此之上是 MetaMorpho 金库，由**风险策展人**管理。

#### 风险策展人模型

```
风险策展人（专业风控公司/DAO/基金）
  │
  ▼
设计并部署 MetaMorpho 金库
  · 选择金库可借入哪些 Morpho Blue 市场
  · 调整分配权重
  · 设定额外的金库级规则
  │
  ▼
存款人选择特定金库 = 选择特定策展人的风险判断

代表：Gauntlet, Steakhouse, MEV Capital, RE7 Labs
```

2026 年初，策展人金库总存款近 110 亿美元。但激进追求收益的金库已经暴露了风险——某些接受高风险稳定币或薄流动性抵押品的金库遭受了损失。

### Sky（原 MakerDAO）：去中心化央行

Sky 不是从现有池子借出，而是基于抵押品存款**铸造新的稳定币**（USDS）。

- **LitePSM**：固定汇率兑换窗口（USDS/DAI ↔ USDC），维持 1 美元锚定
- **Sky 储蓄利率（SSR）**：治理调整利率以影响 USDS 持有需求
- 日益以国债等实物资产为后盾，融合 DeFi 与传统金融

### Wildcat：链上机构信用

Wildcat 走完全相反的路——**有意低抵押**，依赖储备比率流动性缓冲和信用关系。

```
借款人设定所有参数：固定 APR、锁定期、提款窗口
  · 无协议层承销
  · 可要求借贷人签署链下法律协议
  · 储备低于要求 → 市场违约 + 罚款累积

2025 年 Kinto 违约：借贷人遭受 24% 损失——
  但损失仅限于该特定设施，未传染至 Wildcat 其他 1.5 亿+ 贷款
```

## 四、收益生成与优化

### Ethena：Delta 中性生息美元

```
用户存入 ETH/BTC/LST → 铸造 USDe
                         │
             ┌───────────┼───────────┐
             │           │           │
         质押收益     资金费率收入   储备资产收益
        （底层抵押品）（做空永续合约）（国库类资产）
             │           │           │
             └───────────┴───────────┘
                         │
              质押 USDe → sUSDe（复合收益）
```

**核心风险**：
- 资金费率反转——熊市中负费率可能侵蚀收益
- 托管风险——依赖 CEX 进行对冲（通过 OES 提供商部分缓解）
- 锚定偏离——2025 年 10 月 USDe 在 Binance 上一度跌至 $0.62（场所特定事件，链上池保持 ~$1）

### Pendle：交易时间本身

Pendle 将生息资产拆分为两个可交易组件：

```
生息资产（如 stETH、sUSDe）
         │
    ┌────┴────┐
    │         │
  PT（本金代币）  YT（收益代币）
  │              │
  到期后兑换底层  到期前的所有收益
  （类似零息债券）  （收益率杠杆暴露）

数学关系：PT 价格 + YT 价格 ≈ 底层资产价格
```

**PT 循环策略**：存 PT 为抵押品 → 借稳定币 → 买更多 PT → 重复。4-5 倍杠杆下曾产生高双位数回报。但**上行封顶、下行开放**：如果借贷成本超过 PT 隐含收益率，策略可能转为亏损。PT 流动性可能远低于持仓规模——被迫平仓时面临严重价格冲击。

### 积分耕作（Points Farming）

在代币发行前参与协议，累积"积分"——押注未来空投价值。

- 收益完全取决于协议未来决策——规则可能中途更改
- 成功案例（Hyperliquid、Ethena、Usual 空投）验证了策略潜力
- 本质是对协议成功和公平分发的双重赌注

### 期权金库：系统化权利金收取

| 类型 | 机制 | 风险 |
|------|------|------|
| 备兑看涨金库 | 存入 ETH/BTC，卖出虚值看涨期权 | 强势上涨时标的被行权——上行被截断 |
| 现金担保卖出看跌金库 | 持有稳定币，卖出看跌期权 | 下跌时被迫以行权价购入标的 |

收益来自期权权利金，波动率越高权利金越丰厚。但波动率崩塌可能让近期收益的保护变得不充分。

## 五、基础设施依赖

DeFi 的所有精巧机制都建立在共享基础设施之上。预言机故障、闪电贷攻击和桥接漏洞造成的损失，历史上远超核心协议逻辑的缺陷。

### 预言机网络

智能合约无法直接访问外部数据——这就是**预言机问题**。

| 预言机 | 模式 | 特点 |
|--------|------|------|
| **Chainlink** | 推送模式（OCR） | 偏差阈值 + 心跳更新；占据主导地位 |
| **Pyth** | 拉取模式 | 应用按需获取最新价格，适合高吞吐链 |
| **RedStone / Band** | 替代方案 | 提供冗余和不同架构 |

#### 三种预言机配置及其风险

| 配置 | 机制 | 核心风险 |
|------|------|---------|
| **基本面预言机** | 从内部汇率/链上会计推导价格 | 抵抗 DEX 操纵，但流动性危机时产生坏账 |
| **硬编码预言机** | 固定价格关系（如 USDe = USDT ≈ $1） | 脱锚时清算失效，坏账累积 |
| **市场化预言机** | 从 DEX/聚合 feed 获取实际交易价格 | 可能被操纵（尤其浅池），但头寸可被真实清算 |

> 存入任何借贷协议前，先了解其对每种资产的预言机配置。预言机设计决定了你面对的是操纵风险、坏账风险，还是两者的某种组合。

### 闪电贷：双刃剑

```
同一笔交易内：
  借入 → 使用 → 归还（+ 少量费用）
  │
  如果归还失败 → 整笔交易回滚，好像从未发生

合法用途：                  攻击向量：
· 无资本套利               · 操纵薄池价格 → 在借贷协议中套利
· 原子抵押品置换           · 放大重入攻击
· 清算（借入→清算→卖出抵押品）· 治理攻击（借入投票权）
· 跨协议再融资
```

**闪电贷不创造漏洞——它们放大已有漏洞。** 协议必须在假设攻击者拥有大量资本的前提下确保安全。

防御层：检查-效果-交互模式、重入守卫、多区块 TWAP、Chainlink 等独立数据源、隔离模式 + 债务上限 + 借贷上限、滑点检查。

## 章节要点速览

1. **DeFi 的核心力量是原子可组合性**——借款、兑换、存入可在单笔交易内完成，不可能在传统金融中实现
2. **AMM 用数学公式取代了订单簿撮合引擎**——恒定乘积公式保证永远有流动性，代价是大额交易的滑点
3. **Uniswap v3 的集中流动性是资本效率革命**——甚至在稳定币交易上挑战了 Curve 的专业地位
4. **超额抵押是无信任借贷的基石**——Aave 等协议不能起诉违约者，必须有足够资产可供清算
5. **风险策展人模型让存款人选择信任谁**——Morpho 和 Euler 的隔离金库将风险决策从协议层移到策展人层
6. **Ethena 将稳定币发行从被动支持变为主动收益策略**——但引入资金费率反转和托管集中风险
7. **预言机是 DeFi 的阿喀琉斯之踵**——预言机故障造成的损失远超核心协议 Bug
8. **闪电贷放大已有漏洞**——协议安全设计必须假设攻击者有无限资本
